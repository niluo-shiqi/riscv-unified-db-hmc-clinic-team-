# Documentation Site Planning

This directory contains the planning documents for the UDB Docusaurus documentation site.

| File | Purpose |
|---|---|
| `documentation-implementation-plan.md` | **Active working document.** Detailed phase-by-phase implementation plan with per-task status tracking. Start here. |
| `documentation-plan.md` | Content plan — what pages exist, what they cover, and what source material feeds them. |
| `documentation-site-design.md` | Design decisions — landing page layout, navigation structure, audience targeting, and technical notes. |
| `decisions.md` | Decision log — records significant decisions, their rationale, what was implemented as a result, and how to reverse each decision if needed. |

These documents will eventually be consolidated into `docs/contributing/docs-architecture.md` on the live site (see task 9.8 in the implementation plan), at which point this directory can be archived.
